# Plan — Template Premium : Cabinet d'Avocats

## Overview
Build a premium, market-ready law firm website template (Sector 03: Cabinet d'Avocats) using the web-template-library skill. The template must embody legal excellence, authority, and trust — with dual B2B/B2C adaptability.

## Stage 1 — Design System & Architecture
- **Skill**: web-template-library (Phase 1-2)
- **Tasks**:
  1. Define color palette (Navy/White/Gold for business law variant)
  2. Define typography (serif premium headings + sans-serif body)
  3. Define spacing, layout grid, animation patterns
  4. Map out full page tree and component inventory
  5. Create detailed page-level design specs
- **Output**: Complete design system + architecture document

## Stage 2 — Asset Generation
- **Skill**: generate_image
- **Tasks**:
  1. Generate hero background images (2 variants)
  2. Generate team member portraits (8 lawyers)
  3. Generate office/ambiance images (3 images)
  4. Generate icons/illustrations as needed
- **Output**: All visual assets in /public/images/

## Stage 3 — Template Development
- **Skill**: vibecoding-webapp-swarm
- **Tasks**:
  1. Initialize Next.js + Tailwind project
  2. Build global layout (nav, footer, design system)
  3. Build Homepage (hero + expertise cards + stats + team preview + CTA)
  4. Build Expertises page (hub + 8 practice areas)
  5. Build Équipe page (filterable team grid + individual profiles)
  6. Build Publications page (articles hub with filters)
  7. build Actualités page (news & victories)
  8. Build Contact page (dual form: urgent vs planned)
  9. Build Carrières page (culture + open positions)
  10. Build À propos page (history + values + governance)
- **Output**: Complete React/Next.js application

## Stage 4 — Polish & Deploy
- **Tasks**:
  1. Add smooth scroll animations (IntersectionObserver)
  2. SEO metadata + Schema.org markup
  3. Responsive optimization
  4. Accessibility (WCAG AA) check
  5. Build and deploy
- **Output**: Live deployed template
